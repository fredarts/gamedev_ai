# Конфигурация (Получение ключа API)

У Gamedev AI нет проприетарного бэкенда для кражи ваших данных. Он делает прямые вызовы к серверу LLM, который вы предпочитаете использовать. В целях безопасности все проиндексированные знания хранятся локально на вашей машине!

Чтобы плагин мог взаимодействовать с вашим кодом и отвечать, нам необходимо предоставить **ключ API**.

## Использование инструментов Google (Gemini) — Бесплатно
Самый популярный, чрезвычайно умный и **бесплатный** способ использования плагина на данный момент — через Google AI Studio с Gemini 3.1 Pro или Gemini 3.1 Flash. Gemini 4.0 сейчас находится в стадии бета-тестирования.

1. Перейдите в [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Войдите в систему, используя свой аккаунт Google (кредитная карта для бесплатного уровня не требуется).
3. Нажмите **"Get API key"** или **"Create API key"**.
4. Скопируйте сгенерированный длинный ключ `(например: AIzaSyD......)`.

## Привязка ключа в редакторе

1. Внутри панели Gamedev AI в Godot нажмите на скрытую вкладку сверху под названием **"Settings"**.
2. Убедитесь, что в выпадающем меню выбран провайдер **"Gemini"**.
3. Вставьте ваш полный ключ в текстовое поле **API Key**, убедившись, что нет лишних пробелов.
4. В поле *Model Name* введите модель, которую вы предпочитаете (по умолчанию обычно `gemini-3.1-pro` или `gemini-3.1-flash`).

## Использование OpenRouter или OpenAI

Если вы предпочитаете использовать такие модели, как Claude Opus 4.7, GPT-5.5 или другие современные открытые модели, Gamedev AI поддерживает нативно совместимые API.

1. Зарегистрируйтесь и получите свой API-ключ на:
   * [OpenRouter](https://openrouter.ai/keys)
   * [OpenAI](https://platform.openai.com/api-keys)
2. Вернитесь в скрытую вкладку **Settings** внутри Godot.
3. Измените провайдера в выпадающем меню с `Gemini` на `OpenAI / OpenRouter`.
4. Вставьте свой **API Key**.
5. В дополнительном поле под названием **Base URL** введите **точно** соответствующий адрес:
   * **OpenRouter:** `https://openrouter.ai/api/v1`
   * **OpenAI:** `https://api.openai.com/v1`
6. Измените поле **Model Name** на нужный идентификатор модели (например, `anthropic/claude-opus-4.7` или `gpt-5.5`).

## Использование NVIDIA NIM (Бесплатный DeepSeek и другие)

Платформа NVIDIA NIM обеспечивает быстрый и зачастую бесплатный доступ ко многим топовым моделям, таким как серии **DeepSeek** и **Llama**, через OpenAI-совместимый API.

1. Перейдите в [каталог API NVIDIA](https://build.nvidia.com/) и создайте учетную запись, чтобы получить API-ключ.
2. На вкладке плагина **Settings** выберите провайдера **`OpenAI / OpenRouter`**.
3. Вставьте ваш сгенерированный NVIDIA **API Key**.
4. В поле **Base URL** необходимо ввести точный URL NVIDIA NIM:
   * `https://integrate.api.nvidia.com/v1`
5. В поле **Model Name** введите название нужной модели.

> **⚠️ Внимание: Название модели в NVIDIA**
> В NVIDIA NIM вы **обязаны** указывать «префикс организации» перед названием модели.
> Пример: Для DeepSeek нельзя использовать просто `deepseek-v4-pro`, вместо этого нужно писать **`deepseek-ai/deepseek-v4-pro`** (или `deepseek-ai/deepseek-r1`). Для Llama это будет `meta/llama-3.1-70b-instruct`. Если префикс организации не указан, API вернет **ошибку 404 — Not Found**.

## Использование локальных моделей (Ollama / LM Studio)

Предпочитаете запускать модели ИИ прямо на своем компьютере, не полагаясь на внешние API и совершенно бесплатно?

1. Установите [Ollama](https://ollama.ai) и загрузите модель через терминал: `ollama pull llama3.1`
2. Убедитесь, что Ollama запущена (через `ollama serve` или просто оставьте приложение открытым).
3. На вкладке **Settings** измените провайдера на **`Local (Ollama / LM Studio)`**.
4. Поле **Base URL** будет автоматически заполнено значением `http://localhost:11434/v1`.
5. Введите название загруженной модели в поле **Model Name** (например, `llama3.1`, `gemma4`, `deepseek-v4-pro`).
6. Поле API Key будет автоматически отключено (локальным моделям ключ не требуется).

> **⚠️ Примечание:** Скорость и качество ответов полностью зависят от аппаратного обеспечения вашего компьютера (видеокарты) и выбранной модели.
