# Установка (Installation)

Добро пожаловать в официальную документацию **Gamedev AI**! Эта страница посвящена тому, как быстро добавить помощника в ваш текущий проект Godot 4.6.

## Предварительные требования (Prerequisites)
* **Godot Engine:** Версия 4.6 или выше (не требует нативной поддержки `.gdextension`, плагин работает на чистом GDScript).
* Аккаунт Google или OpenRouter для ключей API.

## Подробные шаги для ручной установки (GitHub)

1. Создайте или откройте существующий проект в Godot 4.6.
2. Перейдите в корневую папку вашего проекта (каталог, содержащий файл `project.godot`).
3. Создайте папку с именем `addons` (если она еще не существует).
4. Внутри `addons/` скачайте или клонируйте репозиторий плагина:
```bash
git clone git@github.com:fredarts/gamedev_ai.git
```
*(Или скачайте файл `.zip` со вкладки Releases на GitHub и распакуйте его в `res://addons/gamedev_ai`).*

## Активация в редакторе (Activation)

Как только файлы окажутся на месте:
1. Откройте редактор проекта Godot.
2. Перейдите в меню `Project` сверху и выберите `Project Settings`.
3. Перейдите на вкладку `Plugins` сверху.
4. Найдите **Gamedev AI** в списке установленных плагинов.
5. Установите галочку рядом с `Enabled`.

Вот и все! При первой активации вы увидите, что панель чата Gamedev AI появилась в боковом *доке* вашего редактора.
