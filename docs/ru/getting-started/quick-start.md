# Быстрый старт (Основы)

Все настроено? Давайте теперь применим этого умного помощника на практике.

## Ваш первый запрос (First Prompt)

1. После настройки ключа API во вкладке `Settings` снова нажмите на вкладку **Chat** (в верхнем меню или, если она свернута, на иконку чата).

![Главный интерфейс чата Gamedev AI](../../images/main_interface.png)

2. Gamedev AI был обучен в основном для *понимания кода* и *манипулирования узлами*. Чтобы просто пообщаться, введите ваше пожелание в поле в нижней части панели и нажмите `Shift + Enter` (или кнопку **Send**).
3. Напишите: *"Привет, кто ты? И что ты можешь создать для меня в этом проекте?"*

ИИ ответит через несколько секунд форматированным текстом, эмодзи и даже практическими предложениями по коду.

## Всплывающие меню и встроенные блоки кода

Вы можете не просто делать Ctrl+C в чате. Если скрипт предлагает фрагмент кода `(например: func jump())`, рядом с кодом появятся визуальные кнопки:
- **Copy:** Копирует код в буфер обмена (`Ctrl+V`).

Если вы не знаете, какой формат скрипта ему дать, просто выделите текст внутри любого открытого файла `.gd` в Godot. В плагине Gamedev AI реализованы быстрые команды рефакторинга, применяемые к скрипту, такие как кнопки *Refactor* или *Explain logic*.
