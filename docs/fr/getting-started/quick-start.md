# Démarrage Rapide (Les Bases)

Tout est configuré ? Utilisons maintenant l'assistant intelligent en pratique.

## Votre premier prompt

1. Une fois la clé API configurée dans l'onglet `Settings`, cliquez à nouveau sur l'onglet **Chat** (dans le menu supérieur ou sur l'icône de chat, si elle est réduite).

![Interface principale de Chat dans Gamedev AI](../../images/main_interface.png)

2. Gamedev AI est essentiellement formé pour *Comprendre le Code* et *Modifier les Nœuds*. Pour discuter naturellement, saisissez simplement ce que vous voulez dans le champ inférieur du panneau et appuyez sur `Shift + Enter` (ou cliquez sur **Envoyer**).
3. Tapez : *"Bonjour, qui es-tu et que peux-tu créer pour moi dans ce projet ?"*

L'IA répondra en quelques instants avec du texte stylisé, des emojis ou même en suggérant des blocs de code pratiques.

## Menu flottant et extraits en ligne

Vous n'avez pas seulement besoin de faire Ctrl+C dans le chat. Si un script propose un bloc de code `(par exemple, func jump())`, des boutons visuels apparaîtront à côté du code :
- **Copier :** Copie dans votre presse-papiers (`Ctrl+V`).

Et si vous ne savez pas quel format de script lui donner, sélectionnez un morceau de texte dans n'importe quel fichier `.gd` ouvert dans Godot. Le plugin Gamedev AI dispose de raccourcis de refactorisation visuelle à appliquer aux scripts, tels qu'un bouton *Refactoriser* ou *Expliquer la logique*.
