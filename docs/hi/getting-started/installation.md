# इंस्टॉलेशन (Installation)

**Gamedev AI** के आधिकारिक दस्तावेज़ों में आपका स्वागत है! यह पृष्ठ इस बात पर केंद्रित है कि आप अपने वर्तमान Godot 4.6 प्रोजेक्ट में सहायक को तेज़ी से कैसे जोड़ सकते हैं।

## पूर्वापेक्षाएँ (Prerequisites)
* **Godot Engine:** संस्करण 4.6 या उच्चतर (नेटिव `.gdextension` समर्थन आवश्यक नहीं है, प्लगइन शुद्ध GDScript में चलता है)।
* एपीआई कुंजियों (API Keys) के लिए Google या OpenRouter खाता।

## मैनुअल चरण-दर-चरण (GitHub)

1. Godot 4.6 में एक नया या मौजूदा प्रोजेक्ट बनाएं या खोलें।
2. अपने प्रोजेक्ट के रूट फ़ोल्डर (जहाँ `project.godot` फ़ाइल स्थित है) पर जाएँ।
3. `addons` नाम का एक फ़ोल्डर बनाएं (यदि वह पहले से मौजूद नहीं है)।
4. `addons/` के अंदर, प्लगइन रिपॉजिटरी डाउनलोड या क्लोन करें:
```bash
git clone git@github.com:fredarts/gamedev_ai.git
```
*(या GitHub पर Releases टैब से `.zip` फ़ाइल डाउनलोड करें और इसे `res://addons/gamedev_ai` में निकालें)।*

## एडिटर में सक्रियण (Activation)

एक बार फाइलें अपनी जगह पर होने के बाद:
1. अपने Godot प्रोजेक्ट का एडिटर खोलें।
2. ऊपर `Project` मेनू पर जाएं और `Project Settings` चुनें।
3. ऊपर `Plugins` टैब पर जाएँ।
4. इंस्टॉल किए गए प्लगइन्स की सूची में **Gamedev AI** खोजें।
5. `Enabled` के बगल में स्थित बॉक्स को चेक करें।

बस इतना ही! जब आप इसे पहली बार सक्षम करते हैं, तो आप देखेंगे कि Gamedev AI चैट पैनल आपके एडिटर के दाहिने *Dock* में दिखाई देगा।
