# Mulai Cepat (Quick Start - Dasar)

Semua sudah dikonfigurasi? Mari kita gunakan asisten cerdas ini dalam tindakan nyata.

## Prompt Pertama Anda (First Prompt)

1. Setelah mengonfigurasi kunci API di tab `Settings`, klik kembali pada tab **Chat** (di menu atas atau ikon chat jika diminimalkan).

![Antarmuka chat utama Gamedev AI](../../images/main_interface.png)

2. Gamedev AI dilatih terutama untuk *memahami kode* dan *manipulasi node*. Berinteraksilah secara alami dengan menulis apa yang Anda inginkan di bidang bawah panel dan tekan `Shift + Enter` (atau klik **Send**).
3. Tulis: *"Halo, siapa kamu? Dan apa yang bisa kamu bantu buatkan di proyek saya ini?"*

AI akan menjawab dalam beberapa detik dengan teks terformat, emoji, dan bahkan saran kode nyata.

## Menu Melayang dan Blok Kode Bawaan

Anda tidak hanya bisa menekan Ctrl+C di chat. Jika AI menyarankan segmen kode skrip `(contoh: func jump())`, tombol visual akan muncul di samping kode:
- **Copy :** Menyalin kode ke clipboard (`Ctrl+V`).

Jika Anda tidak tahu format skrip apa yang harus diberikan kepadanya, sorot teks di dalam file `.gd` apa pun yang terbuka di Godot. Ada pintasan refaktor visual yang diterapkan pada skrip di dalam plugin Gamedev AI, seperti tombol *Refactor* atau *Explain logic*.
