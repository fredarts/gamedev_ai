# 快速开始 (基础知识)

一切都配置好了吗？现在让我们在实践中使用这个智能助手。

## 您的第一个提示

1. 在 `Settings` 选项卡中配置好 API 密钥后，再次点击 **Chat** 选项卡（在顶部菜单中，或者如果它被最小化了，点击聊天图标）。

![Gamedev AI 中的主聊天界面](../../images/main_interface.png)

2. Gamedev AI 主要经过训练来 *理解代码* 和 *修改节点*。要自然地聊天，只需在面板下方的字段中输入您的愿望，然后按 `Shift + Enter`（或点击 **Send**）。
3. 输入：*"你好，你是谁？你能在这个项目中为我创建什么？"*

AI 将在几秒钟内以样式化的文本、表情符号甚至实用的代码建议做出响应。

## 悬浮菜单和内联代码片段

您不仅仅可以在聊天中按 Ctrl+C。如果某个脚本提出了一个代码块建议 `(例如：func jump())`，代码旁边会出现可视化按钮：
- **Copy :** 将代码复制到剪贴板 (`Ctrl+V`)。

如果您不知道该给它什么脚本格式，只需在 Godot 中打开的任何 `.gd` 文件中突出显示一段文本。Gamedev AI 插件具有应用于脚本的可视化重构快捷方式，例如 *Refactor* 或 *Explain logic* 按钮。
