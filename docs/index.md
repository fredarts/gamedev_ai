features:
  - title: Construção via Chat
    details: Descreva a mecânica, anexe arquivos e arraste Nodes direto da Scene Tree para o painel de conversa para a IA estruturar o sistema pra você.
  - title: Diff Seguro & Visual
    details: A IA não quebra seu código às cegas. O plugin exibe uma janela lado-a-lado (Antes/Depois) para você aplicar apenas as linhas que aprovar.
  - title: Modo Vigiar (Auto-Depuração)
    details: Deixe a IA vigiar a janela de Output! Se o seu jogo crashar no "Play", ela imediatamente sugere o conserto do script que causou a falha.
  - title: Indexação Vetorial (RAG)
    details: O "Index Codebase" escaneia seus scripts em um Vector DB local. A IA acha as suas classes automaticamente na hora de programar.
  - title: 25 Habilidades Embutidas
    details: O assistente foi treinado com 25 expertises (Shaders, Multiplayer, UI/UX, Inventários) implementando os melhores padrões de mercado.
  - title: Git Sem Complicação
    details: Aba nativa para gerir GitHub. Crie linhas alternativas (Branches), resgate projetos e deixe a IA ler seus scripts para auto-gerar mensagens de Commit claras.
