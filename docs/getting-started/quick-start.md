# Primeiro Uso (O Básico)

Tudo configurado? Agora vamos usar o assistente inteligente na prática.

## O Seu Primeiro Prompt

1. Com a API Key configurada na aba `Settings`, clique novamente na aba **Chat** (no menu superior ou no ícone da conversa, caso encolhido).
2. O Gamedev AI é treinado essencialmente para *Entender Códigos* e *Modificar Nós*. Para conversar naturalmente, basta digitar o que quer no campo inferior do painel e apertar `Shift + Enter` (ou dar clique em **Enviar**).
3. Escreva: *"Olá, quem é você e o que consegue criar neste projeto pra mim?"*

A IA irá se agrupar e responder em instantes com textos estilizados, emojis ou até mesmo sugerindo blocos de código práticos.

## Menu Flutuante e Snippets In-Line

Você não precisa apenas dar Ctrl+C no chat. Se um script te propor um bloco de código `(ex: func pulo())`, haverá botões visuais ao lado do código:
- **Copy:** Copia para a sua prancheta (`Ctrl+V`).

E se você não sabe que formato de script dar pra ele, selecione um texto em qualquer arquivo aberto `.gd` no Godot. O plugin Gamedev AI tem atalhos visuais de refatoração para aplicar nos scripts, como botão de *Refactor* ou *Explicar Lógica*.
