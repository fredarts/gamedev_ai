# ইনস্টলেশন (Installation)

**Gamedev AI** এর অফিসিয়াল ডকুমেন্টেশনে স্বাগতম! এই পেজটি আপনার বর্তমান Godot 4.6 প্রজেক্টে অ্যাসিস্ট্যান্টটিকে দ্রুত যুক্ত করার বিষয়ে আলোকপাত করে।

## প্রয়োজনীয় শর্তাবলী (Prerequisites)
* **Godot Engine:** ভার্সন ৪.৬ বা তার বেশি (নেটিভ `.gdextension` সাপোর্টের প্রয়োজন নেই, প্লাগিনটি বিশুদ্ধ GDScript এ কাজ করে)।
* API কী-এর জন্য Google বা OpenRouter অ্যাকাউন্ট।

## টিউটোরিয়াল: ম্যানুয়াল ইনস্টলেশন (GitHub)

১. Godot 4.6 এ একটি প্রজেক্ট তৈরি করুন বা খুলুন।
২. আপনার প্রজেক্টের রুট ফোল্ডারে যান (যেখানে `project.godot` ফাইলটি রয়েছে)।
৩. `addons` নামে একটি ফোল্ডার তৈরি করুন (যদি ইতিমধ্যে না থাকে)।
৪. `addons/` এর ভেতরে প্লাগিন রিপোজিটরি ডাউনলোড বা ক্লোন করুন:
```bash
git clone git@github.com:fredarts/gamedev_ai.git
```
*(অথবা GitHub এর Releases ট্যাব থেকে `.zip` ফাইল ডাউনলোড করে `res://addons/gamedev_ai` এ এক্সট্রাক্ট করুন)।*

## এডিটরে অ্যাক্টিভেশন (Activation)

ফাইলগুলি যথাস্থানে থাকার পর:
১. আপনার Godot প্রজেক্ট এডিটর খুলুন।
২. উপরের `Project` মেনুতে যান এবং `Project Settings` নির্বাচন করুন।
৩. উপরের `Plugins` ট্যাবে যান।
৪. ইনস্টল করা প্লাগিনগুলির তালিকায় **Gamedev AI** খুঁজে বের করুন।
৫. `Enabled` এর পাশের বক্সে টিক দিন।

ব্যাস! প্রথমবার অ্যাক্টিভেট করার সময় আপনার এডিটরের সাইড *Dock* এ Gamedev AI চ্যাট প্যানেলটি উপস্থিত হতে দেখবেন।
