# التثبيت (Installation)

مرحبًا بك في وثائق **Gamedev AI** الرسمية! تركز هذه الصفحة على كيفية إضافة المساعد بسرعة إلى مشروع Godot 4.6 الحالي الخاص بك.

## المتطلبات الأساسية (Prerequisites)
* **محرك Godot:** الإصدار 4.6 أو أحدث (لا يتطلب دعمًا أصليًا لـ `.gdextension` ، يعمل الملحق بـ GDScript الخالص).
* حساب Google أو OpenRouter لمفاتيح API.

## خطوات يدوية مفصلة (GitHub)

1. قم بإنشاء أو فتح مشروع موجود في Godot 4.6.
2. انتقل إلى المجلد الجذري لمشروعك (المجلد الذي يحتوي على ملف `project.godot`).
3. قم بإنشاء مجلد باسم `addons` (إذا لم يكن موجودًا بالفعل).
4. داخل `addons/` ، قم بتحميل أو استنساخ مستودع الملحق:
```bash
git clone git@github.com:fredarts/gamedev_ai.git
```
*(أو قم بتحميل ملف `.zip` من علامة تبويب Releases في GitHub واستخرجه في `res://addons/gamedev_ai`).*

## التفعيل في المحرر (Activation)

بمجرد وجود الملفات في مكانها:
1. افتح محرر مشروع Godot الخاص بك.
2. انتقل إلى قائمة `Project` في الأعلى واختر `Project Settings`.
3. انتقل إلى علامة تبويب `Plugins` في الأعلى.
4. ابحث عن **Gamedev AI** في قائمة الملحقات المثبتة.
5. حدد المربع بجوار `Enabled`.

هذا كل شيء! عند التفعيل لأول مرة، سترى لوحة دردشة Gamedev AI تظهر في *Dock* الجانبي لمحررك.
